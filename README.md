This project is directly inspired by the popular youtube series JetLag - https://www.youtube.com/@jetlagthegame

Its a game of hide and seek played acrosss entire countries. There are multiple seakers and they ask questions to narrow down the search for the hider,
there are multiple rounds played, switching out the hider each time and whomever has the largest time before being caught is deemed the winner.

This project represents the Client used to render the current location and visualisze the mechanics of the game.
